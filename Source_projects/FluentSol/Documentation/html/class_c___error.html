Opis szczegółowy

Klasa do obsługi błędów.

Klasa umożliwia przechowanie dowolnej ilości komunikatów o błędach a następnie nagranie ich na dysk. Maksymalna długość komunikatu jest limitowana definicją MAX_ERROR w definicje.h
