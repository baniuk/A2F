Opis szczegółowy

Przechowuje zbiór podstawowych parametrów. Enum musi się zgadzać z przechowywanymi parametrami.

Wartości nazw_parametrów musi się zgadzać z przechowywanymi danymi. Parametry zdefiniowane w C_Parameters_Set.h Dodatkowo przechowuje informację czy w danym polu jest dana aktualna czy nie.
