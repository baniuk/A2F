Opis szczegółowy

Klasa przechowuje kontainery z parametrami dla poszczególnych wejsc, parametry powiązane z danymi uzyskanymi z ASPENA oraz ze skryptu. Jedyna różnica pomiędzy klasą C_Parameter_Container.h jest taka, że ta klasa przechowuje obiekty klasy C_Parameter_Container.h, dzięki czemu można przechowywać kilka wejść.

Klasa przechowująca parametry dla poszczególnych składników strumienia dla poszczególnych wejść. Powiązana z ASPENEM, wypełniana głównie w klasie C_myRunProcess.h. Powiązana jest także z parametrami ze skryptu. Niektóre parametry w tej klasie są brane ze skryptu też.
