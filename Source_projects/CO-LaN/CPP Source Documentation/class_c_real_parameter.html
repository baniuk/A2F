Detailed Description

Real parameter class.

CAPE-OPEN Collection class that implements a real parameter. A CAPE-OPEN parameter gives a reference to a ParameterSpecification object; the parameter specification of this parameter is implemented by the parameter itself. So this object implements a real parameter, a parameter specification and a real parameter specification interface (and derives from the CAPE-OPEN base object to implement identification and error interfaces)

We ensure that the current value of the parameter is always valid. This way the implementation of the Validation is trivial
