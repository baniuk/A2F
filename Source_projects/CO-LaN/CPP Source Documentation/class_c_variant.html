Detailed Description

wrapper class for VARIANT values

This class is for Variant data manipilation. The main data type is a VARIANT.

The VARIANT is cleaned up at the destruction of this class, unless it is detected that the VARIANT is owned by somebody else. This is when the class is created from an input variable, or when a return value is requested

For performance, the class is completely inline.

Please take great care of knowing who owns the content of a Variant and therefore who is responsible for freeing the content; this shows from [in], [in, out] and [out, retval] annotations in the IDL
