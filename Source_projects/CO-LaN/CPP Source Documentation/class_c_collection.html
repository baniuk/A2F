Detailed Description

Generic CAPE-OPEN Collection class.

CAPE-OPEN Collection class for exposing a collection of CAPE-OPEN objects. The items in the class are stored as references to CAPEOPENBaseObjects, so that they can be cast to the proper type and so that their name is immediately accesible
