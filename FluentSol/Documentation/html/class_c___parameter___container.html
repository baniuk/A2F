Opis szczegółowy

Klasa jest kontainerem z parametrami opisującymi poszczególne materiały, parametry powiązane z danymi uzyskanymi z ASPENA oraz ze skryptu.

Klasa przechowująca parametry dla poszczególnych składników strumienia. Powiązana z ASPENEM, wypełniana głównie w klasie C_myRunProcess.h. Powiązana jest także z parametrami ze skryptu. Niektóre parametry w tej klasie są brane ze skryptu też.
